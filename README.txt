Этот скрипт позволяет привязать разные кнопки (находящиеся в разных местах сайта) для вызова модального окна.


Использование:

1. Базовая разметка для HTML:
  
	<div class="container" id="about">
		<div class="description-btn">
			Узнать подробнее
		</div>
		<div class="description-btn">
			Узнать подробнее
		</div>
		<div class="description-btn">
			Узнать подробнее
		</div>
		<div class="description-btn">
			Узнать подробнее
		</div>
		/-/-/-/-/-/-/-/-/ (прочая разметка)
		<button class="more"> Узнать больше</button>
	</div>
// кнопки могут быть на разных уровнях вложенности, так как будет использоваться делегирование событий.

2. JS скрипт:

	1) создаються переменные в которые помещаються элементы со страницы:
	2) создаеться функция которая показывает модальное окно и не дает прокручиваться окну (когда модальное окно открыто)
	3) вешается обработчик событий на родителя, чтобы делегировать события на дочерние элементы, на которые мы будем кликать,
	чтобы все сработало правильно и именно те элементы отработали, которые мы хотим делаем проверку и запускаем функцию со второго
	пункта (здесь указано два класса в которых содержаться элементы, но если хотите можете добавить любые классы, которые есть у элементов,
	которые вы хотите, чтобы тоже отрабатывали при клике на них, главное, чтобы они были дочерними (вложенными) в
	<div class="container" id="about"> ... </div>. А так же добавляется эффект с классом more-splash с CSS (можете убрать если у вас не будет
	эффекта при клике.
	4) далее вещается обработчик событий на элемент "х" в модальном окне, который закрывает модальное окно, удаляет класс more-splash,
	если он есть у элементов. И удаляет стиль у document.body.style.overflow = 'hidden'; чтобы окно браузера могло прокручиваться.
